## 報工系統
- 目的:
  1. 實際運作於傳統工廠中
  2. 過去案子是以 php 與 html 進行撰寫
  3. 現以 react + express 框架進行重構
  4. 練習與加強前後端技術
  5. 進行自我提升
- 功能:
  1.  進行現有製令單號報工
  2.  各工單中可以進行人員調動
  3.  人員產線的計時
  4.  完成報工數量統計
  5.  匯出日期區間報表
  6.  設定密碼
  7.  設定休息時間
  8.  進行已完工工單維護作業
- 部屬方式:
  -   docker-compose up --build -d
  -   目前於 http://34.31.84.39/ 進行作業
-   使用技術
  -   React.js
  -   Express.js
  -   PostgresSQL
  -   Docker
  -   Nginx
  -   Postman
  -   pgAdmin
